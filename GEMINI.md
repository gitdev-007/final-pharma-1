# Mediruit Lifesciences Project Instructions

## Project Overview
This is a static landing page for Mediruit Lifesciences. 

## Tech Stack
- **HTML5**: Standard markup.
- **Tailwind CSS**: Currently using the CDN version (`https://cdn.tailwindcss.com`).
- **JavaScript**: Minimal `main.js` for DOM interactions.
- **Assets**: Video backgrounds stored in `assets/videos/`.

## Development Guidelines
- **Styling**: Continue using Tailwind CSS utility classes. Avoid adding custom CSS unless absolutely necessary; if needed, add it to the `<style>` block in `index.html` or a dedicated CSS file.
- **Interactivity**: Use `main.js` for any client-side logic.
- **Mobile First**: Ensure all changes are responsive and follow the Tailwind `sm`, `md`, `lg`, `xl` breakpoints.

## Architecture
- `index.html`: Main entry point.
- `main.js`: Main JavaScript logic.
- `assets/`: Directory for images, videos, and other static assets.
